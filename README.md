##Custom admin panel for vvgcarparts PHP web-application

**Used technologies:**

🔧Platform - .NET via C#

🔧Data transfer - ODBC MySQL Driver

🔧DataBase - MySQL

🎨Visual - WinForms😁

🔧IDE - VS Studio 2017

**Features:**

🔧View full catalog of auto parts

🔧Sort catalog for price and relevance

🔧Login/Registration

🔧Roles: 1. ADMIN for add/update/delete/save reports and 2. USER for basic info display

🔧Catalog is another repository on git
